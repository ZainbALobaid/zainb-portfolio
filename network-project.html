Network Course Project

Computer Networks · CS360 · Saudi Electronic University

Cisco Packet Tracer IP Addressing LAN / WAN OSPF Routers and Switches
Overview

Project Overview

This academic project focused on designing and configuring a multi-LAN network topology using Cisco Packet Tracer. It was completed as part of the Computer Networks course (CS360) at Saudi Electronic University.

Objective

Project Objective

To build a functional network topology, configure devices with appropriate IP settings, connect LANs through routers and switches, configure OSPF dynamic routing, and test communication within and between LANs.

Tools Used

Tools and Concepts

Cisco Packet Tracer LAN / WAN IP Addressing Default Gateway DNS Routers Switches OSPF Dynamic Routing Connectivity Testing
Implementation

Implementation Summary

1
Created three LANs, each containing two end devices connected to a switch.
2
Connected each switch to a dedicated router to serve as the gateway for each LAN.
3
Assigned IP addresses, subnet masks, default gateways, and DNS settings to all devices.
4
Connected the three routers through a central router to form a WAN.
5
Configured OSPF dynamic routing on all routers to enable cross-LAN communication.
6
Tested connectivity using ping commands within each LAN and between different LANs.

Project Outcome
